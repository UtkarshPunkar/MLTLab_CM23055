# 🚀 Machine Learning Technology Lab (MLTLab_CM23055)

Welcome to my **Machine Learning Technology Lab Repository**.

This repository contains browser-based Machine Learning practicals implemented using **TensorFlow.js**, focusing on building, training, evaluating, and deploying models directly in the web environment.

The objective of this course is to provide hands-on experience with modern ML techniques while understanding how models can run efficiently inside a browser.

---

## 👨‍🎓 About:

- **Course:** Machine Learning Technology Lab  
- **Repository:** MLTLab_CM23055  
- **Technology Stack:** JavaScript + TensorFlow.js  
- **Platform:** Web Browser
